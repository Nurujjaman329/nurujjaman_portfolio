MyKrishi

Project Info

Developed MyKrishi, a mobile application designed with dedicated panels for Investors and Agents, aimed at bridging the gap between urban investors and rural farmers through transparent, real-time agricultural project updates.

Key Responsibilities:
  • Designed and implemented separate user panels for Investors and Agents to ensure role-based workflows.
  • Enabled Investors to view detailed project information and make payments directly through the mobile app.
  • Empowered Agents to communicate with farmers and update projects with real-time images and progress reports from the field.
  • Integrated in-app chat functionality to facilitate seamless communication between Investors and Agents.
  • Focused on building trust and transparency in agricultural investments through regular, visual project updates.
  • Integrated secure payment features for seamless transactions.
